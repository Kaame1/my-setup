# nvim

Конфиг Neovim (раскладка «Sethy»). Вся папка симлинкуется в `~/.config/nvim`.

> **Нужен Neovim 0.10+** (используются `vim.lsp.config`/`vim.lsp.enable`,
> snacks.nvim и т.д.). `setup.sh` ставит свежую сборку из стабильного PPA.
>
> **Лидер (leader) = `Space` (пробел).**

## Структура

```
init.lua                     точка входа
lazy-lock.json               зафиксированные версии плагинов (lazy.nvim)
shortcuts.txt                личные заметки
lua/current-theme.lua        активная тема (gruvbox)
lua/Sethy/
├── lazy.lua                 бутстрап lazy.nvim + импорт плагинов
├── core/
│   ├── options.lua          опции редактора
│   └── keymaps.lua          все базовые кеймапы
└── plugins/                 по файлу на плагин
    ├── lsp/mason.lua        Mason + список LSP
    └── lsp/lspconfig.lua    настройка LSP + кеймапы on_attach
```

## Первый запуск

При первом старте `nvim` **lazy.nvim** ставит себя и все плагины из
`lazy-lock.json`, затем **Mason** ставит LSP-серверы и инструменты:
`lua_ls`, `bashls`, `pyright`, `clangd`, `html`, `cssls`, `tailwindcss`,
`angularls`, `emmet_ls`, `marksman`, плюс `prettier`, `stylua`, `isort`,
`pylint`. Для них нужны `node`, `python3` и C-тулчейн — их ставит `setup.sh`.

## Быстрый гайд — горячие клавиши

### Редактирование / перемещение

| Клавиши            | Действие                                          |
| ------------------ | ------------------------------------------------- |
| `J` / `K` (visual) | сдвинуть выделенные строки вниз / вверх            |
| `J` (normal)       | приклеить строку снизу, курсор на месте            |
| `Ctrl+d` / `Ctrl+u`| полстраницы вниз/вверх, курсор по центру           |
| `n` / `N`          | к следующему/предыдущему совпадению, по центру      |
| `<` / `>` (visual) | отступ влево/вправо, не теряя выделение             |
| `<leader>p` (x)    | вставить поверх выделения, не затирая буфер          |
| `<leader>Y`        | скопировать строку в системный буфер                 |
| `<leader>d`        | удалить в «чёрную дыру» (не трогая буфер)            |
| `x`                | удалить символ, не трогая буфер                     |
| `Ctrl+c`           | в insert — Esc; в normal — снять подсветку поиска    |
| `<leader>s`        | заменить слово под курсором по всему файлу          |
| `<leader>x`        | сделать текущий файл исполняемым (работает и в Oil)  |
| `<leader>fp`       | скопировать путь файла в буфер                      |
| `Ctrl+f`           | открыть новый gnome-terminal                        |

### Окна и вкладки

| Клавиши                        | Действие                    |
| ------------------------------ | --------------------------- |
| `<leader>sv` / `<leader>sh`    | сплит вертикально / горизонтально |
| `<leader>se` / `<leader>sx`    | выровнять сплиты / закрыть сплит  |
| `<leader>sm`                   | развернуть/свернуть сплит (Maximizer) |
| `<leader>to` / `<leader>td`    | новая / закрыть вкладку      |
| `<leader>tn` / `<leader>tp`    | следующая / предыдущая вкладка|

### Поиск файлов (Snacks picker / Telescope)

| Клавиши         | Действие                          |
| --------------- | --------------------------------- |
| `<leader>pf`    | найти файлы                       |
| `<leader>ps`    | grep по проекту                   |
| `<leader>pws`   | grep слова под курсором/выделения |
| `<leader>pr`    | недавние файлы                    |
| `<leader>pk`    | поиск по кеймапам                 |
| `<leader>ex`    | файловый explorer                 |
| `<leader>vh`    | страницы help                     |
| `<leader>th`    | сменить цветовую тему             |

### LSP

| Клавиши      | Действие                 |
| ------------ | ------------------------ |
| `gd` / `gD`  | к определению / объявлению|
| `gi` / `gr`  | имплементации / ссылки   |
| `K`          | подсказка (hover)        |
| `<leader>rn` | переименовать            |
| `<leader>ca` | code action              |
| `<leader>f`  | форматировать буфер      |

### Git (fugitive + gitsigns + snacks)

| Клавиши        | Действие                    |
| -------------- | --------------------------- |
| `<leader>gg`   | статус (`:Git`)             |
| `<leader>lg`   | LazyGit                     |
| `<leader>gs` / `<leader>gr` | stage / reset hunk (в т.ч. в visual) |
| `<leader>gp`   | предпросмотр hunk           |
| `<leader>gbl` / `<leader>gB` | blame строки / вкл-выкл blame |
| `<leader>gd`   | diff файла                  |
| `<leader>wl` / `<leader>wc` | git worktree: список / создать |

### Файлы (Oil) и прочее

| Клавиши        | Действие                              |
| -------------- | ------------------------------------- |
| `-`            | открыть родительскую папку в Oil      |
| `<leader>-`    | Oil в плавающем окне                  |
| `q` (в Oil)    | закрыть Oil                           |
| `<leader>u`    | Undotree (дерево отмен)               |
| `gcc` / `gc`   | закомментировать строку / выделение (mini.comment) |
| `sa` / `ds` / `sr` | добавить / удалить / заменить обрамление (mini.surround) |
| `sj` / `sk`    | join / split аргументов (mini.splitjoin) |
| `<leader>cw`   | стереть висячие пробелы               |
| `]t` / `[t`    | к следующему / предыдущему TODO       |

Дополнительные заметки — в `shortcuts.txt`.
