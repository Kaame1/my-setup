# terminator

Конфиг терминала Terminator.

- **Файл:** `config` → симлинк на `~/.config/terminator/config`
- **Тема:** Gruvbox Dark (16 цветов), прозрачный фон (`0.98`), Monospace 13,
  буфер прокрутки 10k строк (бесконечный), копирование по выделению.

## Профили

| Профиль   | Фон             | Команда                | Для чего                     |
| --------- | --------------- | ---------------------- | ---------------------------- |
| `default` | тёмный `#282828`| `/usr/bin/fish`        | обычный терминал             |
| `claude`  | кремовый `#FDF5E3` | `~/.claude/run-claude` | Claude Code в светлой теме |

## Быстрый гайд — горячие клавиши

### Разбиение и панели

| Действие                | Клавиши              |
| ----------------------- | ------------------- |
| Разбить горизонтально   | `Ctrl+Shift+O`      |
| Разбить вертикально     | `Ctrl+Shift+E`      |
| Переход между панелями  | `Alt + стрелки`     |
| Изменить размер панели  | `Ctrl+Alt + стрелки`|
| Закрыть панель          | `Ctrl+Shift+W`      |
| Развернуть панель (zoom)| `Ctrl+Shift+Z`      |

### Вкладки

| Действие              | Клавиши                          |
| --------------------- | -------------------------------- |
| Новая вкладка         | `Ctrl+Shift+T`                   |
| Следующая / предыдущая| `Ctrl+Shift+N` / `Ctrl+Shift+M`  |
| Перейти к вкладке 1–5 | `Alt+1` … `Alt+5`                |

### Прочее

| Действие              | Клавиши                          |
| --------------------- | -------------------------------- |
| Копировать / вставить | `Ctrl+Shift+C` / `Ctrl+Shift+V`  |
| Поиск                 | `Ctrl+Shift+F`                   |
| Масштаб +/-/сброс      | `Ctrl+=` / `Ctrl+-` / `Ctrl+0`  |
| Сгруппировать всё      | `Super+G` / `Super+Shift+G`     |
| Полный экран          | `F11`                            |
| Закрыть окно          | `Ctrl+Shift+Q`                   |

> Профиль `claude` запускает `~/.claude/run-claude`; путь автоматически
> переписывается в `setup.sh`, если имя пользователя другое.
